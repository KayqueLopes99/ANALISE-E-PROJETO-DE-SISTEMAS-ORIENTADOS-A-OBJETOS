
📋 User Stories - Sistema Restaurante Mix (Híbrido)
Este documento contém as histórias de usuário para o sistema de gestão de restaurante que contempla os modelos Presencial e Delivery.

🔹 Módulo: Pedido Geral (Base)
 * US01 - Visualização de Preparo Como Cozinheiro,
   Eu quero visualizar os itens de um pedido e seu status,
   Para que eu possa organizar a sequência de preparo dos pratos na cozinha.
 * US02 - Manutenção de Cardápio Como Gerente,
   Eu quero cadastrar e atualizar itens no cardápio (nome e preço),
   Para que os valores dos pedidos estejam sempre corretos e atualizados.
 * US03 - Controle de Fluxo Como Atendente,
   Eu quero alterar o status de um pedido (Pendente → Preparando → Pronto),
   Para que o cliente ou o entregador saiba exatamente quando o prato foi finalizado.

   
🔹 Módulo: Atendimento Presencial
 * US04 - Identificação de Mesa Como Garçom,
   Eu quero abrir um pedido informando o número da mesa,
   Para que eu saiba exatamente para onde levar os pratos e bebidas.
 * US05 - Cálculo de Taxa de Serviço Como Cliente de Mesa,
   Eu quero que o sistema calcule automaticamente a taxa de serviço (10%) sobre o subtotal,
   Para que eu tenha transparência e agilidade no fechamento da conta.
 * US06 - Adição de Itens Como Garçom,
   Eu quero adicionar novos itens a um pedido de mesa já existente,
   Para que o cliente possa solicitar rodadas extras de consumo sem fechar a conta atual.


🔹 Módulo: Delivery (Entrega)
 * US07 - Registro de Destino Como Cliente Delivery,
   Eu quero informar meu endereço de entrega ao realizar o pedido,
   Para que o restaurante saiba o local exato para o envio da mercadoria.
 * US08 - Atribuição de Logística Como Operador de Caixa,
   Eu quero atribuir um entregador específico a um pedido de delivery,
   Para que eu possa rastrear qual profissional está responsável por cada entrega.
 * US09 - Cálculo de Frete Como Cliente Delivery,
   Eu quero que o sistema some a taxa de entrega ao valor total dos produtos,
   Para que eu saiba o custo total da conveniência antes de finalizar a compra.
